12 Thì Tiếng Anh

Present: Hiện tại | Past: Quá khứ | Future: Tương lai

1. Present Simple - Hiện Tại Đơn
Form / Cấu tạo:
+ : S + V(s/es) / am/is/are
+ : Chủ ngữ + V(s/es) / am/is/are
- : S + do/does not + V
- : Chủ ngữ + do/does not + V
? : Do/Does + S + V?
? : Do/Does + chủ ngữ + V?
Uses / Cách dùng:
  • Habits, routines, repeated actions
  • Thói quen, hành động lặp lại
  • General truths, facts, scientific laws
  • Sự thật hiển nhiên, quy luật
  • States, timetables
  • Trạng thái, lịch trình cố định
always, usually, often, every day/week, never...
Examples / Ví dụ:
• She plays tennis every weekend.
• Cô ấy chơi tennis mỗi cuối tuần.
• The Earth revolves around the Sun.
• Trái Đất quay quanh Mặt Trời.
2. Present Continuous - Hiện Tại Tiếp Diễn
Form:
+ : S + am/is/are + V-ing
+ : Chủ ngữ + am/is/are + V-ing
- : S + am/is/are not + V-ing
- : Chủ ngữ + am/is/are not + V-ing
? : Am/Is/Are + S + V-ing?
? : Am/Is/Are + chủ ngữ + V-ing?
Uses:
  • Actions happening right now
  • Hành động đang xảy ra lúc nói
  • Temporary situations
  • Tình huống tạm thời
  • Changing situations
  • Tình huống đang thay đổi
  • Future arrangements
  • Kế hoạch đã sắp xếp
now, at the moment, right now, currently, look!, listen!...
Examples:
• I am studying English now.
• Tôi đang học tiếng Anh bây giờ.
• We are flying to Hanoi tomorrow.
• Chúng tôi bay đến Hà Nội ngày mai.
3. Present Perfect - Hiện Tại Hoàn Thành
Form:
+ : S + have/has + V3/ed
+ : Chủ ngữ + have/has + V3/ed
- : S + have/has not + V3/ed
- : Chủ ngữ + have/has not + V3/ed
? : Have/Has + S + V3/ed?
? : Have/Has + chủ ngữ + V3/ed?
Uses:
  • Experiences up to now
  • Kinh nghiệm đến nay
  • Actions with present result
  • Hành động có kết quả hiện tại
  • Unfinished time (this week...)
  • Thời gian chưa kết thúc
ever, never, just, already, yet, since, for...
Examples:
• I have visited Paris twice.
• Tôi đã đến Paris hai lần.
• She has lived here since 2010.
• Cô ấy sống ở đây từ năm 2010.
4. Present Perfect Continuous - Hiện Tại Hoàn Thành Tiếp Diễn
Form:
+ : S + have/has been + V-ing
+ : Chủ ngữ + have/has been + V-ing
- : S + have/has not been + V-ing
- : Chủ ngữ + have/has not been + V-ing
? : Have/Has + S + been + V-ing?
? : Have/Has + chủ ngữ + been + V-ing?
Uses:
  • Duration from past to now
  • Thời lượng từ quá khứ đến nay
  • Recent actions with visible effects
  • Hành động gần đây để lại dấu vết
for, since, all day, lately...
Examples:
• It has been raining all morning.
• Trời mưa cả buổi sáng rồi.
5. Past Simple - Quá Khứ Đơn
Form:
+ : S + V2/ed / was/were
+ : Chủ ngữ + V2/ed / was/were
- : S + did not + V
- : Chủ ngữ + did not + V
? : Did + S + V?
? : Did + chủ ngữ + V?
Uses:
  • Completed actions in past
  • Hành động hoàn tất quá khứ
  • Past habits
  • Thói quen quá khứ
yesterday, last..., ago, in 1990...
Examples:
• He visited his family last weekend.
• Anh ấy thăm gia đình cuối tuần trước.
6. Past Continuous - Quá Khứ Tiếp Diễn
Form:
+ : S + was/were + V-ing
+ : Chủ ngữ + was/were + V-ing
- : S + was/were not + V-ing
- : Chủ ngữ + was/were not + V-ing
? : Was/Were + S + V-ing?
? : Was/Were + chủ ngữ + V-ing?
Uses:
  • Actions in progress at past time
  • Hành động đang diễn ra lúc đó
  • Interrupted actions
  • Hành động bị gián đoạn
at... yesterday, while, when...
Examples:
• I was eating when she called.
• Tôi đang ăn thì cô ấy gọi.
7. Past Perfect - Quá Khứ Hoàn Thành
Form:
+ : S + had + V3/ed
+ : Chủ ngữ + had + V3/ed
- : S + had not + V3/ed
- : Chủ ngữ + had not + V3/ed
? : Had + S + V3/ed?
? : Had + chủ ngữ + V3/ed?
Uses:
  • Action before another past action
  • Trước một hành động quá khứ khác
before, after, by the time...
Examples:
• They had left before we arrived.
• Họ đã đi trước khi chúng tôi đến.
8. Past Perfect Continuous - Quá Khứ Hoàn Thành Tiếp Diễn
Form:
+ : S + had been + V-ing
+ : Chủ ngữ + had been + V-ing
- : S + had not been + V-ing
- : Chủ ngữ + had not been + V-ing
? : Had + S + been + V-ing?
? : Had + chủ ngữ + been + V-ing?
Uses:
  • Duration before past action
  • Thời lượng trước hành động quá khứ
for, since before past time...
Examples:
• She had been working all day, so she was tired.
• Cô ấy đã làm việc cả ngày, nên mệt.
9. Future Simple - Tương Lai Đơn
Form:
+ : S + will + V
+ : Chủ ngữ + will + V
- : S + will not + V
- : Chủ ngữ + will not + V
? : Will + S + V?
? : Will + chủ ngữ + V?
Uses:
  • Predictions, spontaneous decisions
  • Dự đoán, quyết định lúc nói
  • Promises, offers
  • Lời hứa, đề nghị
tomorrow, next..., soon, I think...
Examples:
• It will rain tomorrow.
• Ngày mai trời sẽ mưa.
10. Future Continuous - Tương Lai Tiếp Diễn
Form:
+ : S + will be + V-ing
+ : Chủ ngữ + will be + V-ing
- : S + will not be + V-ing
- : Chủ ngữ + will not be + V-ing
? : Will + S + be + V-ing?
? : Will + chủ ngữ + be + V-ing?
Uses:
  • Actions in progress at future time
  • Hành động đang diễn ra lúc tương lai
at this time tomorrow...
Examples:
• At 8 AM tomorrow, I will be studying.
• Mai lúc 8h sáng tôi đang học.
11. Future Perfect - Tương Lai Hoàn Thành
Form:
+ : S + will have + V3/ed
+ : Chủ ngữ + will have + V3/ed
- : S + will not have + V3/ed
- : Chủ ngữ + will not have + V3/ed
? : Will + S + have + V3/ed?
? : Will + chủ ngữ + have + V3/ed?
Uses:
  • Completed before future time
  • Hoàn thành trước thời điểm tương lai
by tomorrow, by the time...
Examples:
• By 2030, I will have finished university.
• Đến 2030, tôi sẽ đã tốt nghiệp đại học.
12. Future Perfect Continuous - Tương Lai Hoàn Thành Tiếp Diễn
Form:
+ : S + will have been + V-ing
+ : Chủ ngữ + will have been + V-ing
- : S + will not have been + V-ing
- : Chủ ngữ + will not have been + V-ing
? : Will + S + have been + V-ing?
? : Will + chủ ngữ + have been + V-ing?
Uses:
  • Duration up to future point
  • Thời lượng đến thời điểm tương lai
for... by the time...
Examples:
• By next month, she will have been living here for 5 years.
• Tháng sau, cô ấy sẽ đã sống ở đây 5 năm.
